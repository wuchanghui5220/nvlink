什么是 InfiniBand？
InfiniBand (IB) 是专为高性能计算 (HPC)、人工智能 (AI)、大规模数据中心和企业级应用设计的计算机网络通信标准。它不是对传统以太网的简单升级，而是一套从零开始构建的、以低延迟、高带宽和高效率为核心目标的交换网络架构。
🚀 核心技术：远程直接内存访问 (RDMA) - 实现极致性能的关键
📡
RDMA 核心技术
Remote Direct Memory Access
允许一台计算机的处理器直接读写另一台计算机的内存，无需占用各自操作系统的处理时间。通过将通信开销从 CPU 卸载到专用的网络硬件（通道适配器），实现极低延迟和极高吞吐量。
⚡
极致性能表现
Ultra-Low Latency & High Throughput
链路速率持续演进：HDR (200Gb/s)、NDR (400Gb/s) 甚至 XDR (800Gb/s)。成为驱动全球顶级超级计算机和 AI 训练集群的首选互联技术。
🛡️
原生无损网络
Lossless Network Architecture
通过复杂的基于信用的端到端流量控制机制，从根本上防止因接收方缓冲区不足而导致的数据包丢失。这对于需要确定性性能的 HPC 和 AI 任务至关重要。
🌐
完整技术生态
End-to-End Solution
提供从芯片、网卡、交换机到管理软件和上层通信库的端到端优化解决方案。集中式子网管理器 (SM) 统一负责网络拓扑发现、LID 分配和路由表计算与下发。
