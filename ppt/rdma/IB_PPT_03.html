网络组件
基础架构与关键术语
网络组件 · 节点类型 · 地址体系 · 管理架构
🖥️
节点 (Node)
Processing Node / IO Unit
包括处理器节点 (HCA) 和 IO 单元 (TCA)。是网络中的计算或存储实体。
🔌
通道适配器 (CA)
Channel Adapter
节点与 InfiniBand 网络的接口硬件，负责所有通信协议处理和RDMA操作。
🔀
交换机 (Switch)
InfiniBand Switch
在子网内部根据 LID 转发数据包，提供高速无阻塞交换能力。
🌐
路由器 (Router)
InfiniBand Router
根据 GID 连接不同子网，实现跨子网通信。
🏠
子网 (Subnet)
InfiniBand Subnet
由一个 SM (子网管理器) 管理的设备集合，是IB网络的基本管理单元。
🏷️
本地 ID (LID)
Local Identifier
16 位地址，子网内唯一，用于高效路由转发。
🌍
全局 ID (GID)
Global Identifier
128 位地址，基于 IPv6 格式，用于跨子网通信。
🏗️
InfiniBand 网络架构
InfiniBand 采用交换网络架构，通过交换机互联各个节点，提供无阻塞的高性能通信能力。每个组件都有明确的职责和功能。

双重地址体系
L
本地标识符 (LID)
• 16位地址空间
• 子网内唯一
• 高效硬件路由
• 支持65536个端口
• 由SM动态分配管理
G
全局标识符 (GID)
• 128位地址空间
• 全局唯一
• IPv6格式兼容
• 跨子网通信必需
• 支持路由器转发
