NVUE CLI 命令

NVUE CLI 命令 (在 Cumulus Linux 交换机上运行)

⚙
基础配置管理
nv config show
显示当前的 NVUE 配置
nv config show -o commands
以 CLI 命令的形式显示当前的 NVUE 配置
🌉
网桥和MLAG管理
nv show bridge domain br_default
验证 br_default 网桥的配置
nv show mlag
验证 MLAG 的操作状态
nv show interface --view=mlag-cc
验证 MLAG 接口状态，并检查 MLAG 对等体之间是否存在配置冲突
nv show mlag consistency-checker global
验证两个 MLAG 对等体之间是否存在全局配置冲突
🔗
VXLAN和BGP管理
nv show nve vxlan
验证 NVE 接口 (VTEP) 的状态和配置
nv show vrf default router bgp neighbor
显示 BGP 邻居的详细信息，包括自治系统、会话状态、正常运行时间、收发的前缀计数等
📡
EVPN VNI管理
nv show evpn vni <VNI_ID>
验证特定 EVPN VNI 的条目（例如：nv show evpn vni 10）
nv show evpn vni -o raw
以原始格式显示 EVPN VNI 的信息，包括VNI类型、VXLAN接口、MAC地址数量、ARP数量、远程VTEP数量和租户VRF
nv show evpn vni
显示所有 EVPN VNI 的摘要信息，包括MAC地址数量、ARP数量和远程VTEP数量
🔍
MAC地址表管理
nv show bridge domain br_default mac-table
验证 br_default 网桥上学习到的 MAC 条目
nv show evpn vni <VNI_ID> mac
验证特定 VNI 的 MAC 地址条目是否被填充到 EVPN 中（例如：nv show evpn vni 10 mac）

使用技巧

  • 使用 -o raw 参数获取详细原始信息
  • 使用 --view 参数查看特定视图
  • 命令支持Tab键补全功能
  • 可使用 | 管道符过滤输出
💡 快捷提示
大多数 nv show 命令都支持添加具体参数来查看详细信息。

重要说明

  • NVUE CLI 是只读接口，用于查看状态
  • 配置修改需要通过配置文件或API
  • 支持JSON输出格式便于自动化
  • 命令执行需要适当权限

输出格式选项

  • -o json - JSON格式输出
  • -o yaml - YAML格式输出
  • -o raw - 原始详细格式
  • -o commands - CLI命令格式
