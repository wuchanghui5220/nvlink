基础软件与驱动

为 NVIDIA 网络硬件提供稳定、高性能的底层支持。这一层是发挥硬件全部潜能的基石，确保最高的吞吐量和最低的延迟。

    核心组件

  • MLNX_OFED: 包含适配器所需的所有驱动、固件工具和库的官方软件包
  • Adapter Firmware: 提供最新的固件版本以启用新功能、修复错误并优化性能
  • NVIDIA-Peer Memory: GPU Direct RDMA 支持，实现 GPU 间高速数据传输
  • 驱动程序更新: 定期更新确保与最新硬件和操作系统的兼容性

    支持平台

  • Linux (RHEL, Ubuntu, SLES)
  • Windows Server
  • VMware ESXi / KVM
  • FreeBSD
构筑了高性能网络的坚实地基

网络操作系统与管理

提供开放、灵活的网络操作系统和强大的集中管理平台，通过自动化和深度可视性，极大简化数据中心网络的运维复杂度。

    核心组件

  • Cumulus Linux: 业界领先的开放式网络操作系统，让交换机像服务器一样易于管理
  • NetQ: 提供实时的网络状态验证与故障排查，贯穿网络部署全生命周期
  • UFM Platform: 强大的 InfiniBand 集群管理平台，支持大规模 AI 集群
  • NVIDIA Air: 免费的云端网络仿真平台

    关键特性

  • 自动化网络配置与部署
  • 实时网络监控和告警
  • 网络拓扑可视化
  • 预测性故障分析
实现了网络管理的自动化与智能化

加速与卸载平台

通过革命性的 DPU 硬件和 DOCA 软件开发套件，将网络、存储、安全等基础设施任务从 CPU 卸载，释放宝贵的计算资源以加速应用。

    核心组件

  • NVIDIA DOCA: 一站式软件开发框架，解锁 BlueField DPU 的强大功能
  • BlueField DPU: 数据中心基础设施处理器，集成 ARM 核心、网络和加速引擎
  • DOCA Services: 预构建的基础设施服务（防火墙、负载均衡等）
  • DOCA Libraries: 丰富的开发库和 API

    应用场景

  • 网络虚拟化 (OVS, OVN)
  • 存储加速 (NVMe-oF)
  • 安全服务 (加密、防火墙)
  • AI 数据处理管道优化
重新定义了数据中心的基础架构
