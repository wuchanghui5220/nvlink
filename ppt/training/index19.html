操作系统选择
**Ubuntu:** 最受欢迎的选择，拥有庞大的社区支持，与NVIDIA驱动、CUDA及主流AI框架（TensorFlow, PyTorch）兼容性极佳。
**RHEL/Rocky Linux:** 在企业环境中因其稳定性和长期支持而备受青睐。NVIDIA同样提供官方支持。
Ubuntu 20.04/22.04 LTS (kernel 5.4+/5.15+)
RHEL/Rocky Linux 8/9 (kernel 4.18+/5.14+)
⚙️
基础配置与命令
**系统更新:** 保持系统最新是安全和稳定的基础。
**安装编译工具:** 安装驱动和CUDA Toolkit需要`gcc`等编译工具。
**关闭图形化界面:** 对于专用的计算服务器，图形化界面会占用宝贵的GPU资源。
# Ubuntu/Debian: sudo apt update && sudo apt full-upgrade -y
# 设置默认启动模式为文本模式: sudo systemctl set-default multi-user.target
系统优化配置
🔧
系统服务配置
启用必要的系统服务，禁用不需要的后台进程以释放系统资源。
配置自动启动服务和系统监控组件，保证系统稳定运行。
建立标准化的服务管理和监控机制。
# 禁用不必要的服务: systemctl disable firewalld
# 启用GPU持久化服务: systemctl enable nvidia-persistenced
🛡️
安全配置
配置SSH安全访问策略，建议禁用root直接登录和密码认证。
设置防火墙和访问控制策略，保护系统安全。
建立用户权限管理和审计日志机制，保证系统安全可追溯。
# SSH配置: PermitRootLogin no
# 禁用密码认证: PasswordAuthentication no
