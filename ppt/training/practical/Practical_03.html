BIOS设置配置
1. 必要设置项目
1. 启用 IOMMU / VT-d
虚拟化和GPU直通必需功能
2. CPU 性能模式
将CPU频率策略设置为性能模式以获得最佳性能
3. 关闭 ACS（Access Control Services）
为了NVIDIA GPU和网卡的正常工作
2. 验证命令
检查 IOMMU 状态
dmesg | grep -i iommu
[ 11.287745] DMAR-IR: IOAPIC id 8 under DRHD base 0x9c3fc000 IOMMU 19
[ 16.839506] iommu: Default domain type: Translated
[ 16.839506] iommu: DMA domain TLB invalidation policy: lazy mode
检查 CPU 频率信息
cpupower frequency-info

检查 ACS 状态
lspci -vvv | grep ACSCtl
⚠️ 如果看到 SrcValid+ 等ACS功能启用，需要在BIOS中关闭或使用命令行禁用
3. 运行时关闭ACS（临时方案）
查询并关闭 SrcValid+
# 查询当前ACS状态
lspci -vvv | grep ACSCtl | grep SrcValid+

# 关闭SrcValid+
lspci -vvv | grep 'PCI bridge' | cut -d' ' -f 1 | xargs -I{} setpci -s {} ECAP_ACS+06.w=0000
⚠️ 此方法仅临时有效，重启后需重新执行。建议在BIOS中永久关闭ACS
