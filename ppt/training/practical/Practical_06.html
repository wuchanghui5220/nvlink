GPU驱动和CUDA安装
1. 下载CUDA工具包
下载CUDA安装包
wget https://developer.download.nvidia.com/compute/cuda/13.0.0/local_installers/cuda_13.0.0_580.65.06_linux.run
💡 建议选择runfile格式，便于自定义安装选项
2. 安装前准备
检查是否有默认GPU驱动
lsmod | grep nouveau
如果没有输出证明没有加载nouveau模块。如果有输出，需要禁用
禁用nouveau驱动（如需要）
echo "blacklist nouveau" >> /etc/modprobe.d/blacklist.conf
echo "options nouveau modeset=0" >> /etc/modprobe.d/blacklist.conf
⚠️ 修改后需要重启系统生效
3. 安装CUDA
运行CUDA安装程序
sh cuda_13.0.0_580.65.06_linux.run --silent --driver --toolkit --override-driver-check
--silent: 静默安装 | --driver: 安装驱动 | --toolkit: 安装工具包
配置环境变量
echo "export PATH=/usr/local/cuda-13.0/bin${PATH:+:${PATH}}" >> ~/.bashrc
echo "export LD_LIBRARY_PATH=/usr/local/cuda-13.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}" >> ~/.bashrc
source ~/.bashrc
4. 配置GPU持久化
安装持久化服务
cd /usr/share/doc/NVIDIA_GLX-1.0/samples/
tar -xvf nvidia-persistenced-init.tar.bz2
cd nvidia-persistenced-init/
./install.sh
启动并设置开机自启
systemctl start nvidia-persistenced.service
systemctl enable nvidia-persistenced.service
💡 持久化可以保持GPU状态，提高多进程访问GPU的性能
5. 验证安装
检查CUDA版本
nvcc --version
nvidia-smi
