GPU测试和NCCL测试
1. GPU Burn 压力测试
下载并编译GPU Burn
git clone https://github.com/wilicc/gpu-burn.git
cd gpu-burn
make
指定GPU运行测试
CUDA_VISIBLE_DEVICES=0,1,2,3,4,5,6,7 ./gpu_burn 300
运行300秒GPU压力测试，测试所有8个GPU
监控GPU状态
watch -n 1 nvidia-smi
Tested 8 GPUs:
GPU 0: OK
GPU 1: OK
...
GPU 7: OK
2. NCCL安装与编译
安装OpenMPI
apt-get install openmpi-bin openmpi-common libopenmpi-dev
下载并编译NCCL
sudo apt install build-essential devscripts debhelper fakeroot
git clone https://github.com/NVIDIA/nccl.git
cd nccl
make -j src.build
make pkg.debian.build
dpkg -i build/pkg/deb/*.deb
下载并编译NCCL测试工具
git clone https://github.com/NVIDIA/nccl-tests.git
cd nccl-tests
make MPI=1 MPI_HOME=/usr/lib/x86_64-linux-gnu/openmpi
3. NCCL性能测试
单节点NCCL测试
./build/all_reduce_perf -b 128M -e 8G -f 2 -g 8
或使用mpirun方式：
mpirun --allow-run-as-root -np 8 ./build/all_reduce_perf -b 128M -e 8G -f 2
多节点NCCL测试
首先配置主机文件myhosts：
gpu2 slots=8
gpu4 slots=8
gpu5 slots=8
gpu6 slots=8
提前配置（关闭ACS）：
lspci -vvv | grep ACSCtl | grep SrcValid+
lspci -vvv | grep 'PCI bridge' | cut -d' ' -f 1 | xargs -I{} setpci -s {} ECAP_ACS+06.w=0000
运行多节点测试：
mpirun --allow-run-as-root \
-hostfile myhosts2 \
-np 32 -npernode 8 \
-x LD_LIBRARY_PATH \
-x NCCL_DEBUG=INFO \
-x NCCL_IB_HCA=mlx5_1,mlx5_2,mlx5_4,mlx5_7 \
-x NCCL_SOCKET_IFNAME=bond0 \
./build/all_reduce_perf -b 128M -e 8G -f 2 -g 1
4. 测试注意事项
环境变量配置
💡 NCCL_IB_HCA指定使用的IB网卡，NCCL_SOCKET_IFNAME指定以太网接口
故障排除
⚠️ 如遇到问题，可设置NCCL_DEBUG=INFO查看详细日志信息
