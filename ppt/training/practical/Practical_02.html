操作系统安装步骤
1. 基础软件安装
安装基础工具包
安装命令
apt install -y openssh-server curl wget vim htop tree ansible ipmitool lldpd net-tools unzip
openssh-server
curl
wget
vim

tree
ansible
ipmitool
lldpd

unzip
2. 内核管理（可选）
查询当前内核版本
cat /proc/version
uname -r
uname -a
安装特定内核版本
# 查看可用内核版本
apt search linux-image

# 安装特定版本
sudo apt install linux-image-[版本号]
sudo apt install linux-headers-[版本号]

# 更新引导
sudo update-grub
3. 关闭系统自动更新
修改自动更新配置
sed -i.bak 's/1/0/' /etc/apt/apt.conf.d/10periodic
禁用内核自动更新
vi /etc/apt/apt.conf.d/50unattended-upgrades
添加内容
Unattended-Upgrade::Package-Blacklist {
  "linux-image-*";
  "linux-headers-*";
};
