IB交换机固件升级
1. 前提条件与准备工作
操作位置建议
建议在对应网络（计算、存储）的UFM上进行操作
⚠️ 拥有两个网络的设备上操作时，无法指定对应的网卡，可能无法升级对应网络的交换机
2. 检查交换机固件
查询交换机LID列表
ibswitches
查询单个交换机固件
flint -d lid-496 query
批量查询固件版本
ibswitches | awk '{print $11}' | grep -o '[0-9]\+' | xargs -I {} sh -c 'echo "LID: {}" && flint -d lid-{} query | grep "FW Version"'
💡 这个命令可以快速查看所有交换机的固件版本
3. 升级固件步骤
下载固件文件
wget https://www.mellanox.com/downloads/firmware/fw-Quantum-2-rel-31_2016_1028-MQM9790-NS2X_Ax.signed.bin.zip
unzip fw-Quantum-2-rel-31_2016_1028-MQM9790-NS2X_Ax.signed.bin.zip
刷新固件
flint -d lid-496 -i fw-Quantum-2-rel-31_2016_1028-MQM9790-NS2X_Ax.signed.bin burn
⚠️ 确保固件文件与交换机型号匹配
重启交换机生效
flint -d lid-496 swreset
4. 升级CPLD固件
安装MFT工具
找一台设备安装即可
tar xzf mft-4.xx.x-xxx.tgz
cd mft-4.xx.x-xxx
./install.sh --oem
mst start
mst status
检查CPLD固件版本
CPLD工具需要向代理商获取，一般查看完以后都需要升级到最新的
./updateswitchcpld --unmanaged -d lid-26 --check_cpld --verbose
批量检查CPLD版本
ibswitches | awk '{print $11}' | grep -o '[0-9]\+' | xargs -I {} sh -c 'echo "LID: {}" && ./updateswitchcpld --unmanaged -d lid-{} --check_cpld --verbose'
升级CPLD固件
永远升级最新版即可，没有配套关系
# 不带管理的交换机
./updateswitchcpld --unmanaged -d lid-<lid of switch>
# 带管理的交换机
./updateswitchcpld --managed -t <switch's IP> -u <username> -p <password> --os mlnx-os
重启交换机
flint -d lid-19 swreset
5. 注意事项
升级范围
只能升级不带管理的交换机，升级带管理的需联系代理商
固件版本匹配
交换机要和网卡的固件版本匹配，建议升级到最新的LTS版本
统一性要求
交换机的固件要统一，网络中的网卡固件要统一
