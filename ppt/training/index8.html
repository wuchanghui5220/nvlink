🔍 固件查询
交换机固件查询
ibswitches
查看所有交换机列表和LID信息
Switch : 0xfc6a1c0300cdd480 ports 65 "Compute-SU4-Leaf32-D09-26U" base port 0 lid 496 lmc 0
flint -d lid-496 query
使用LID查询交换机固件版本
网卡固件查询
ibdev2netdev
查看IB设备和网络接口对应关系
mlx5_0 port 1 ==> ib0 (Up)
mlx5_1 port 1 ==> ib1 (Up)
flint -d mlx5_0 query
查询指定网卡的固件版本
⬆️ 固件升级
固件下载链接
https://network.nvidia.com/support/firmware/firmware-downloads/
1
上传固件到服务器
scp fw-Quantum-2-rel-31_2014_2170-MQM9790-NS2X_Ax.signed.bin.zip server@<ip>:/path
2
解压获取bin文件
unzip fw-Quantum-2-rel-31_2014_2170-MQM9790-NS2X_Ax.signed.bin.zip
3
交换机固件升级
flint -d lid-496 -i fw-Quantum-2-rel-31_2014_2170-MQM9790-NS2X_Ax.signed.bin burn
使用LID方式升级交换机固件
4
网卡固件升级
flint -d mlx5_0 -i fw-ConnectX7-rel-28_43_2566-MCX75310AAS-NEA_Ax-UEFI-14.37.13-FlexBoot-3.7.500.signed.bin burn
网卡升级操作方法类似交换机
