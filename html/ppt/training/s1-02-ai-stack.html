2. AI 系统全栈架构
从应用层到基础设施层，AI 系统的完整技术栈
全栈分层架构
应用层 — ChatBot / 图像生成 / 自动驾驶 / 代码助手
框架层 — PyTorch / TensorFlow / JAX / DeepSpeed / Megatron-LM
运行时 — CUDA / cuDNN / NCCL / MPI / RDMA Verbs
驱动层 — GPU Driver / OFED / 固件
硬件层 — GPU / NVLink / NVSwitch / InfiniBand / 存储
基础设施 — 供电 / 散热(液冷) / 机柜 / 布线
计算维度
  • GPU — 万级 CUDA Core + Tensor Core
  • GPU 互联 — NVLink (900GB/s) + NVSwitch
  • CPU — 任务调度、数据预处理
  • 内存 — HBM3e (80~192GB/GPU)
通信维度
  • 机内 — PCIe 5.0 / NVLink / NVSwitch
  • 机间 — InfiniBand NDR 400Gb/s
  • 集合通信 — AllReduce / AllGather / Ring
  • 协议栈 — RDMA / GPUDirect / SHARP
存储维度 & 管理维度
  • 并行文件系统 — Lustre / GPFS / BeeGFS
  • 对象存储 — Checkpoint / Dataset 持久化
  • 本地 NVMe — 训练数据缓存加速
  • 集群调度 — Slurm / Kubernetes
  • 监控运维 — UFM / DCGM / Prometheus
  • 网络管理 — Subnet Manager / QoS
