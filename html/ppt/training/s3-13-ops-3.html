13. 运维管理 — 常见故障处理方法
硬件类故障
故障诊断命令处置
服务器宕机ipmitool power statusBMC 重启 / 报修
GPU 过热降频nvidia-smi -q -d TEMP检查液冷/风扇
GPU 卡离线lspci | grep NVIDIA重置 PCIe / 重插卡
GPU 内存错误nvidia-smi -q -d ECC隔离节点 + 报修
网络类故障
故障诊断命令处置
IB 端口 Downibstat检查线缆 / 光模块
速率降级iblinkinfo更换线缆 / 光模块
SM 未连接sminfo检查 OpenSM 服务
带宽异常ib_write_bw路由 / 拥塞分析
软件/驱动故障
故障诊断处置
驱动不兼容dmesg | grep -i nvidia回退版本 / 重装
RDMA 异常ibv_devinfo重启 openibd
MPI 启动失败检查 SSH 互信修复 hostfile/权限
存储挂载失败mount / dmesg检查存储服务状态
故障预防策略
  • 预防性维护 — 固件定期升级，线缆定期巡检
  • 监控告警 — 温度/ECC/端口异常即时通知
  • 备份冗余 — SM 双活、数据定期备份
  • 文档管理 — 变更记录、故障 KB 积累
  • 演练机制 — 季度故障演练，验证恢复流程
