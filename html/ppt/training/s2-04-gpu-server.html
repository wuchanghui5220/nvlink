4. GPU 服务器概述
核心组件概览
组件角色与配置建议
GPU核心算力单元 — H200，HBM3e 141GB
CPU任务调度 — Xeon/EPYC，每 GPU ≥ 6 物理核
内存≥ 全部显存总和 2x（8×141G → 1128GB+）
存储NVMe SSD 本地缓存 + 并行文件系统
网络IB NDR 400Gb/s × 8 口（每 GPU 一路）
GPU 互联技术
技术带宽特点
PCIe 5.0128 GB/s (x16)CPU↔GPU 基础互联
NVLink 4900 GB/sGPU↔GPU 高速点对点
NVSwitch全互联机内 8 卡全连接无阻塞
运维要点：检查 GPU 均匀分布在不同 CPU Socket 和 PCIe Root Port 下，避免资源争抢
BIOS 关键配置
  • Above 4G Decoding — 必须启用，支持大显存地址空间
  • PCIe Link Speed — 设为 Gen5 最高版本
  • CPU Power Mgmt — Maximum Performance 模式
  • VT-x / SR-IOV — 虚拟化场景按需启用
BMC 远程管理
  • IPMI / Redfish API 远程电源控制与监控
  • 传感器：温度、功耗、风扇转速实时采集
  • GPU 高功耗 — 需重点设置温度告警阈值
